# ARAYA image assets

Place final photography in the folders below. The site reads paths from `app/lib/araya-media.ts`.

If a file is missing, a cinematic gradient displays instead — no broken images.

**AI prompts:** `/ARAYA_MEDIA_PROMPTS.md`

## Folders

| Folder | Purpose |
|--------|---------|
| `hero/` | **One image only** — `araya-hero.jpg` (homepage hero exclusive) |
| `interiors/` | Residence care, optional contact interior |
| `lifestyle/` | Pool terrace, guest lifestyle, operations, ownership |
| `ocean/` | Aerial editorial (manifest; not used as homepage filler) |
| `sunset/` | Signature moment, terrace evening, sunset CTA |
| `../videos/` | `araya-hero.mp4` optional cinematic hero |

## Hero (exclusive)

- `hero/araya-hero.jpg` — used **only** on the homepage hero
- `videos/araya-hero.mp4` — auto-plays when detected (see `HeroBackground.tsx`)
